# Mistyspace

Minimal starter for a personal portfolio built with Next.js.

Getting started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000 to view the app. Edit `src/app/page.tsx` and `src/app/layout.tsx` to begin.

Next steps

- Replace the placeholder page with your site sections (Hero, About, Work, Contact).
- Add assets under `public/` and components under `src/components/`.

If you want, I can scaffold a basic portfolio layout and components next.
