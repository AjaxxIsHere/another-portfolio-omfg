## 🛠️ Local Development Setup

Want to spin this up on your local machine? Follow these steps:

**1. Clone the repository:**
```bash
git clone [https://github.com/AjaxxIsHere/another-portfolio-omfg.git](https://github.com/AjaxxIsHere/another-portfolio-omfg.git)
cd another-portfolio-omfg

```

**2. Install dependencies:**

```bash
npm install

```

**3. Configure Environment Variables:**
Create a `.env.local` file in the root directory and add your API keys:

```env
SENDGRID_API_KEY=your_sendgrid_api_key
UPSTASH_REDIS_REST_URL=your_upstash_redis_rest_url
UPSTASH_REDIS_REST_TOKEN=your_upstash_redis_rest_token
NEXT_PUBLIC_RECAPTCHA_SITE_KEY=your_recaptcha_site_key
RECAPTCHA_SECRET_KEY=your_recaptcha_secret_key

```

**4. Fire up the development server:**

```bash
npm run dev

```

Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) in your browser to see the application running.
